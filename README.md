# Voice-Disorder-Detection-App

## Dataset

The dataset for this project is stored in Google Drive for easy access and fast loading into Google Colab.

**Location:** [Voice_Project_Data - Google Drive Folder](https://drive.google.com/drive/folders/YOUR_FOLDER_ID_HERE)

### Dataset Structure
The Voice_Project_Data folder contains the following subcategories with .wav audio files:
- Normal
- Laryngitis
- Vocal_Cord_Nodules_Polyps
- Muscle_Tension_Dysphonia
- Spasmodic_Dysphonia
- Vocal_Cord_Paralysis

### Accessing the Dataset in Colab
To use this dataset in your Colab notebook:
1. Run `drive.mount('/content/drive')` to connect to your Google Drive
2. The data path will be: `/content/drive/MyDrive/Voice_Project_Data`
